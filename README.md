#### Quick Project startup
in **container**  
- `docker-compose up --build`
- `curl -X localhost:3000`
or **locally**  
- `npm start`

This is a micro-bloggin frontend written in React inspired by Twitter

-[x] is container friendly

-[ ] has style
-[ ] send chat message
-[ ] read chat message

-[ ] login/logout

-[ ] responsive

-[ ] has some fancy auth
